# FINMENT - Product Description

## What is FINMENT?

FINMENT is a **professional appliance repair booking platform** that allows customers to easily book repair services, track their bookings, and manage service requests - all from one place.

## Core Functionality

**For Customers:**
- 🔧 Browse 6 service types (Plumbing, Fridge, AC, Geyser, Washing Machine, Microwave)
- 📋 4-step booking form with photo uploads
- 📱 Real-time booking status tracking
- ⭐ Leave reviews and ratings
- 🔐 Secure account management

**For Technicians:**
- ✅ Assigned job dashboard
- 📍 View customer details
- 🔄 Update job status
- 📊 Track completed repairs

**For Admin:**
- 📑 Manage all bookings
- 👥 Assign technicians to jobs
- 📊 View booking analytics
- ⚙️ Full system control

## Key Features

✅ **Real-time sync** - Live booking updates  
✅ **Mobile responsive** - Works on all devices  
✅ **Secure authentication** - Firebase Auth  
✅ **Photo uploads** - Customers can attach repair images  
✅ **Email notifications** - Automated booking updates  
✅ **Professional UI** - Modern design with smooth animations  
✅ **Fully tested** - 29+ automated tests  
✅ **Production ready** - Live and secure  

## Technology Stack

- **Frontend:** React + Vite
- **Backend:** Firebase (Auth + Firestore)
- **Hosting:** Vercel CDN
- **Security:** Enterprise-grade Firestore rules
- **Database:** Real-time Firestore
